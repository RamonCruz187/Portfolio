Portfolio

Proyecto final del curso Argentina Programa 4.0. Desarrollado con Angular, Spring Boot y MySQL. Es un proyecto full stack en donde el front se realiza con Angular, ademas consume una API desarrollada en Spring Boot, que se comunica con una base de datos MySQL. Tanto el Front, como el Back y la base de datos fueron desplegados independientemente.

Url: https://github.com/RamonCruz187/Portfolio

EduNotify

El objetivo principal de EduNotify es proporcionar una plataforma que fortalezca la comunicación entre directivos escolares y padres de alumnos. Esta herramienta está diseñada para modernizar y optimizar los métodos de comunicación en el ámbito educativo. App desarrollada en conjunto con Nikol Quintero, que se encargó de desarrollar la app en Kotlin, mi trabajo en este proyecto fue desarrollar y desplegar el back y la base de datos.

Url: https://github.com/RamonCruz187/s16-26-kotlin

Encriptador de Texto

Challenge de Alura Latam, objetivos del proyecto: poder encriptar cualquier texto, poder desencriptar cualquier texto encriptado previamente, realizar un diseño responsivo que se adapte a las diferentes pantallas. poner en práctica todo lo desarrollado en el curso, tanto HTML, CCS y Javascript, uso de Figma para poder copiar el diseño de la página y uso de Trello para poder ir siguiendo el orden de las tareas.

Url: https://github.com/RamonCruz187/Encriptador-de-Texto

Gammer Goods

¡Únete a nuestra comunidad de los videojuegos relacionados con sus franquicias y consolas favorita! El objetivo principal de este proyecto es proporcionar una plataforma en línea accesible y fácil de usar para que los fanáticos de los videojuegos puedan comprar productos de merchandising. Proyecto desarrollado en No-Country, mi trabajo en este proyecto fue realizar gran parte del back con Spring Boot, colaborar activamente con el front con Angular, y desplegar tanto el front con Railway, como el Back con Render.

Url: https://github.com/RamonCruz187/c18-26-java-angular

Contacto Mascota

Aplicación web dedicada a exhibir datos acerca de mascotas que han sido encontradas, facilitando el proceso de adopción de animales y permitiendo la publicación de aquellos que se encuentran extraviados. Proyecto desarrollado en la parte del front con Reac, y en back con Java-Spring Boot, mi trabajo en este proyecto fue desarrollar gran parte del back y realizar el despliegue tanto del back, como de la base de datos.

Url: https://github.com/RamonCruz187/c15-22-m-java-react
